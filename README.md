# Spring Boot Finance and Stock Dashboard Backend
Hybrid SQL + MongoDB | Industry-Standard Architecture | Production-Ready Practices

This repository demonstrates a real-world Spring Boot backend application that integrates both Relational (SQL) and NoSQL (MongoDB) databases within a single, well-structured codebase, following industry-standard architecture and best practices.

The project simulates a real-time finance & stock analytics system, showcasing how modern enterprise applications combine:

SQL databases for transactional, strongly consistent financial data

MongoDB for high-throughput, flexible, time-series market data
